#  Leitor de Configurações com Variáveis de Ambiente

Projeto  em Python mostrando o uso de `os.getenv` e arquivos `.env` para carregar configurações externas.

Desenvolvido no nível de **programador júnior**, seguindo boas práticas do mercado para lidar com credenciais e configurações.

---

##  Funcionalidades

✔ Lê variáveis de ambiente usando `os.getenv`  
✔ Usa o pacote `python-dotenv` para carregar um `.env`  
✔ Verifica variáveis obrigatórias  
✔ Imprime configurações carregadas  
✔ Exemplo real de como sistemas lidam com API keys e modos debug
